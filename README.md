# Personal GitHub Website

This repo contains a simple personal website built with plain HTML, CSS, and JavaScript so it can be deployed easily with GitHub Pages.

## Files

- `index.html` - page structure
- `styles.css` - layout and visual styling
- `script.js` - editable content for projects, notes, and contact links

## Publish with GitHub Pages

1. Push this repo to GitHub.
2. In the repository settings, open `Pages`.
3. Set the source to `Deploy from a branch`.
4. Choose the `main` branch and `/ (root)`.
5. Save, then wait for GitHub to publish the site.

## Customize

- Update the text in `index.html` for your name, intro, and about section.
- Edit the arrays in `script.js` to add your real projects, notes, and contact links.
- Replace the hero image URL in `index.html` with a photo that feels more like you.
